 Descripción / Description
 
(Español)
Este proyecto realiza la ingesta, limpieza y almacenamiento de datos sobre la demanda de reservas hoteleras. El dataset original proviene de la investigación de Nuno Antonio et al. (2019), cuya documentación técnica se encuentra en el archivo `README.pdf` adjunto.

(English)
This project performs the ingestion, cleaning, and storage of hotel booking demand data. The original dataset is based on the research by Nuno Antonio et al. (2019), with technical documentation available in the attached `README.pdf` file.

 Estructura / Structure
- `/data`: Contiene el archivo `hotel_bookings.csv` / Contains the `hotel_bookings.csv` file.
- `/src`: Scripts de carga y gestión de BBDD / Data loading and database management scripts.
- `mi_base_de_datos.db`: Base de datos SQLite / SQLite database.

 Instrucciones / Instructions
1. Instalar dependencias / Install dependencies: `pip install -r requirements.txt`
2. Ejecutar script de base de datos / Run database script: `python src/02_db_manager.py`
3. La base de datos ya está lista para el análisis / The database is ready for analysis.
